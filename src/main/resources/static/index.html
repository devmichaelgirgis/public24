Welcome to our demo!

This is a limited demo of our brand new chat bot: Public 24. This app provides an interface to PrivatBank API. It's developed as an open-source pet project by:
  • Kate Kovshar and Michael Horbunov (server side and API.AI)
  • Alexander Horbunov (front-end).

Current features of this bot (v 0.1) include:

  • "Current Exchange Rate" - returns current exchange rate. Is available for cash and non-cash transactions (non-cash by default). Request example: Current cash exchange for USD.
  • "Exchange Rate History" - returns exchange rate history for inputted data. Request example: CAD for 23/5/2001.
